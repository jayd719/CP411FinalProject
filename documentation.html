Documentation: Render 3D Objects with Enhanced Controls

1. Introduction

This project implements a real-time 3D object rendering application using OpenGL. It allows users to render and interact with 3D shapes like cubes, pyramids, spheres, and custom STL models. The application includes features like rotation, zoom, translation, dynamic lighting, and saving the object being rendered as PNG file.

2. Installation

Prerequisites

  1. A C++ compiler supporting OpenGL and GLUT (g++ or gcc)
  2. Windows OS
  3. Command Line Interface like Shell,Command Prompt etc.

Steps

  1. Download the zip file Here
  2. cd into src folder
  3. Build on Windows using command:
    g++ Main.cpp -o render3D -lfreeglut -lComdlg32 -lopengl32 -lglu32
  4. Run the applicaiton using :
    ./render3D

Features

  • Predefined 3D Objects: Includes Cube, Pyramid, and Sphere, with dynamic rendering and interaction.
  • Custom Model Support: Load binary STL files for rendering custom 3D models.
  • Standard Viewpoits: Standard viewpoints include Isometric, Reverse Isometric, Top, Bottom, and Front views, offering comprehensive perspectives for clear visualization.
  • Real-Time Interaction: Rotate, zoom, and translate objects using keyboard and mouse inputs.
  • Right-Click Menu: Access options for resetting, saving screenshots, switching shapes, and exiting the application.
  • Saving As PNG: Capture and save the current frame as a PNG file with a timestamp.
  • Responsive Design: Automatically adjust rendering based on window resizing.
  • Extensible Architecture: Modular design for adding new shapes, features, and enhancements.

Usage Guide

Launching the Application

  1. Run the compiled binary to open the application window.
  2. A cube will be rendered by default, rotating along the axes.

Switching Shapes

  1. Right-click to open the context menu.
  2. Choose between Cube, Pyramid, Sphere, or Load Custom Model.

Loading Custom Models

  1. Select "Load Custom Model" from the menu.
  2. Browse for a binary STL file using the file dialog.
  3. The selected model will be rendered in the scene.

Saving as PNG

  1. Select "Save As PNG" from the menu.
  2. The current frame will be saved with a timestamped filename in the working directory.

Controls

The 3D Object Rendering Application provides enhanced controls for interacting with and viewing 3D objects.

Keyboard Controls

Key Action
+ Increase rotation speed
- Decrease rotation speed
Space Pause/Resume rotation
w Rotate the object upward
s Rotate the object downward
a Rotate the object to the left
d Rotate the object to the right
q Rotate the object counterclockwise along Z-axis
e Rotate the object clockwise along Z-axis
i Set Isometric view (35.26° X-axis, 45° Y-axis)
r Set Reverse isometric view (-35.26° X-axis, -45° Y-axis)
t Set Top view (-90° X-axis)
b Set Bottom view (90° X-axis)
f Set Front view (0° X, Y, Z axes)
p Save the current frame as a PNG file
Esc Exit the application

Mouse Controls

Action Description
Horizontal Movement Rotate the object left or right based on mouse X-axis movement.
Vertical Movement Rotate the object up or down based on mouse Y-axis movement.
Scroll Up/Down Zoom in/out.
Right-click Open the context menu.

Mouse Options

The right-click menu provides the following functionalities:

  • Reset: Reset all transformations to their default values.
  • Pause/Resume: Toggle rotation.
  • Speed: Adjust the rotation speed dynamically.
  • Zoom: Increase or decrease the zoom level.
  • Load Custom Model: Load and render a binary STL model.
  • Save As PNG: Save the current frame as an image file.
  • Exit: Close the application.

Dependencies

  • OpenGL: Core graphics rendering library.
  • GLUT: For windowing and event handling.
  • STB Image Write: Lightweight library for saving PNG images.
